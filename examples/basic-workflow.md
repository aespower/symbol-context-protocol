# Basic SCP Workflow Example

## Natural Language

A user defines a goal. The system converts it into a task, assigns it to an agent, the agent executes, and the task completes.

## SCP-C (canonical — what travels between agents)

```txt
G > T > A > X = OK
```

## SCP-V (visual — what humans see)

```txt
🎯 ↓ 📋 ↓ 🤖 ↓ ⚡ → ✓
```

## Meaning

| SCP-C | SCP-V | Meaning |
|---|---|---|
| `G` | 🎯 | Goal |
| `>` | ↓ | Leads to |
| `T` | 📋 | Task |
| `A` | 🤖 | Agent |
| `X` | ⚡ | Execute |
| `=` | → | Produces |
| `OK` | ✓ | Complete |
