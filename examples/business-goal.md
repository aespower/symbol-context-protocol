# Business Goal SCP Example

## Natural Language

The objective is to grow the project and generate revenue.

## SCP-C

```txt
G = [GROW+REV]
```

## SCP-V

```txt
🎯 → [📈 + 💰]
```

## AIOS Use Case

```txt
SCP-C: M:founder_goal = [GROW+REV]
SCP-V: 🧠(founder_goal): 🎯 → [📈 + 💰]
```

The founder goal stored in memory is growth and revenue.
