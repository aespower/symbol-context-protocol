# Contributing to Symbol Context Protocol (SCP)

Thank you for your interest in contributing to Symbol Context Protocol.

SCP is an early-stage open source proposal for optimizing context in AI systems using symbols, compact semantic structures, and graph-oriented representations.

## Ways to Contribute

- Propose new symbols.
- Improve the grammar.
- Create examples.
- Design parsers.
- Build SDKs.
- Test SCP with LLMs.
- Integrate SCP with AIOS, LangGraph, CrewAI, AutoGen, or other agent frameworks.

## Contribution Principles

1. Semantic clarity.
2. Human readability.
3. LLM compatibility.
4. Compactness.
5. Graph compatibility.

## Symbol Proposal Template

```md
## Symbol Proposal

Symbol:
Meaning:
Category:
Example:
Why this symbol is useful:
```

## Example

```md
Symbol: 🧪
Meaning: Experiment
Category: Research
Example: 🎯 ↓ 🧪 ↓ 📊
Why useful: Represents testing and validation workflows.
```

## Pull Requests

Before submitting a pull request:

- Keep documentation clear.
- Avoid unnecessary complexity.
- Add examples when possible.
- Explain why the change improves SCP.

## Code of Conduct

Be respectful, constructive, and focused on improving the protocol.
